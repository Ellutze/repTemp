Using_repTemp
=============

This page provides instructions on how to use repTemp


Installation
------------
repository can be installed into an environemnt using pip, as per below. (if applicable)

.. code-block:: bash

	pip install git+https://github.com/Ellutze/repTemp.git
	
or with specific version:

.. code-block:: bash

	pip install git+https://github.com/Ellutze/repTemp.git@v0.8.2
